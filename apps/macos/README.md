# Starbeam macOS App

This folder contains the SwiftUI macOS 14+ menu bar app.

## Requirements

- macOS 14+
- Xcode 15+ (recommended)

## Run

1. Open `apps/macos/Starbeam/Starbeam.xcodeproj` in Xcode.
2. Select the `Starbeam` scheme.
3. Run.

## Configure

- Server base URL: `Starbeam -> Settingsâ€¦ -> Server`
- Default: `http://localhost:3000`

The app is designed to run even before backend endpoints exist (it will show a signed-out / empty state).
